# Demo Rails Application Template

This is a sample template file used for a talk given at [BrisRuby](http://www.meetup.com/BrisRuby/) on the 24th of March 2014. 

## What it contains

* devise
* cancan
* haml-rails
* mysql2
* capistrano
* bootstrap-sass
* capybara
* factory girl
* should matchers
* rspec rails
* timecop
* zeus
